# COMP3000_Autonomous_RC_Car


For people interested in autonomous vehicles who would like to explore how the principals being developed for autonomous cars can be utilized for remote control cars. The Autonomous RC Car is a hardware platform controlled by an integrated Raspberry PI with user configuration provided by a web solution.
